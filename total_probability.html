全概率公式

从部分到整体 —— 统计学中由已知条件概率推算总概率的核心工具

Law of Total Probability

什么是全概率公式

在许多实际问题中，事件 B 的发生往往伴随着若干"原因"或"条件"事件 A₁, A₂, …, Aₙ 的出现。我们很难直接计算 P(B)，但如果知道在每个原因 Ai 下 B 发生的概率 P(B|Ai)，以及各原因出现的概率 P(Ai)，就可以通过全概率公式将它们"拼合"出总的 P(B)。

简单理解：全概率公式 = 分情况讨论后加权汇总。它体现了"化整为零、聚零为整"的思想。

公式表达

P(B) = Σ P(Ai) · P(B | Ai)
i = 1, 2, …, n
即   P(B) = P(A₁)P(B|A₁) + P(A₂)P(B|A₂) + … + P(Aₙ)P(B|Aₙ)

公式成立的前提条件

  • 完备性：A₁, A₂, …, Aₙ 两两互不相容（互斥），即 AiAj = ∅（i ≠ j）
  • 不遗漏：ΣAi = Ω（样本空间的完备事件组），即所有原因覆盖了全部可能
  • 正概率：P(Ai) > 0，每个原因都有非零概率

图示理解

事件 B
↑   由各分支汇总
A₁ → P(B|A₁)
权重 P(A₁)
A₂ → P(B|A₂)
权重 P(A₂)
Aₙ → P(B|Aₙ)
权重 P(Aₙ)

每个 Ai 是一个"原因通道"，B 可以通过任何一个通道发生。
全概率公式将各通道的概率按权重 P(Ai) 加起来，得到 P(B) 的总值。

经典例题

例1：工厂产品合格率

某工厂有 3 条生产线生产同一种产品，其产量占比与合格率如下：

生产线产量占比 P(Ai)合格率 P(B|Ai)
A₁（甲线）50%95%
A₂（乙线）30%90%
A₃（丙线）20%85%

求：随机取一件产品，它是合格品（B）的概率。

P(B) = P(A₁)P(B|A₁) + P(A₂)P(B|A₂) + P(A₃)P(B|A₃)
= 0.50 × 0.95 + 0.30 × 0.90 + 0.20 × 0.85
= 0.475 + 0.27 + 0.17
= 0.915，即合格率为 91.5%
例2：医疗诊断概率

某地区人群患某种疾病的比例为 1%（事件 A），不患病的比例为 99%（事件 Ā）。

已知：该病的血液检测对患者的阳性率为 95%（灵敏度 P(B|A)），对非患者的假阳性率为 5%（P(B|Ā)）。

求：随机检测一人呈阳性（B）的总概率。

P(B) = P(A)·P(B|A) + P(Ā)·P(B|Ā)
= 0.01 × 0.95 + 0.99 × 0.05
= 0.0095 + 0.0495
= 0.059，即阳性总概率约 5.9%
延伸思考
阳性总概率仅 5.9%，但真正患病者仅占阳性中的 0.95/5.9 ≈ 16.1% —— 这正是贝叶斯公式要回答的问题。
例3：信号传输错误率

数字通信系统中，信号经 3 条不同信道传输，各信道的使用概率与错误率如下：

信道使用概率 P(Ai)错误率 P(B|Ai)
信道1（光纤）60%0.1%
信道2（微波）30%1%
信道3（铜缆）10%5%
P(B) = 0.60 × 0.001 + 0.30 × 0.01 + 0.10 × 0.05
= 0.0006 + 0.003 + 0.005
= 0.0086，即总错误率约 0.86%

与贝叶斯公式的对比

全概率公式

P(B) = Σ P(Ai)·P(B|Ai)
  • 方向：由原因推结果
  • 已知 P(Ai) 和 P(B|Ai)
  • 求的是总的 P(B)
  • 属于"正向"概率计算

贝叶斯公式

P(Ai|B) = P(Ai)·P(B|Ai) / P(B)
  • 方向：由结果推原因
  • 已知 P(B)（常由全概率公式算出）
  • 求的是特定原因的后验概率
  • 属于"逆向"概率推理
核心关系
全概率公式是贝叶斯公式的前置步骤——贝叶斯公式中的 P(B) 正是通过全概率公式计算得到的。两者常配合使用，构成"先算总概率、再溯原因"的完整推理链。

应用领域

工程与制造

计算多来源、多工序产品的总体合格率、故障率等。如半导体芯片经多道工序后总的良率。

医学与公共卫生

估算筛查检测的阳性率、患病率；配合贝叶斯公式评估检测的真实诊断价值（灵敏度 vs 假阳性）。

通信与信息论

在多信道、多路径传输中，计算总的误码率或丢包率。

金融与保险

不同经济情景下资产违约的总概率；保险精算中多风险因子叠加的总赔付率。

机器学习与数据科学

混合模型（Mixture Model）中观测数据的边际概率正是全概率公式的体现；EM 算法的 E 步依赖此类计算。

常见误区

误区1：忽略完备性条件

若 A₁, A₂, …, Aₙ 不构成完备事件组（遗漏了某些情况），则 ΣP(Ai)·P(B|Ai) 不等于 P(B)。必须确保所有可能的原因都被覆盖。

误区2：混淆互斥与独立

完备事件组要求 Ai 之间互斥（不能同时发生），而非独立。互斥和独立是两个完全不同的概念——事实上互斥事件（P>0）一定不独立。

误区3：直接加条件概率

错误做法：P(B) = P(B|A₁) + P(B|A₂) + …。正确做法必须乘以权重 P(Ai) 后再求和，否则忽略了各原因出现的可能性差异。

公式推导

由事件分解：B = BA₁ ∪ BA₂ ∪ … ∪ BAₙ
因 Ai 互斥，故 BAi 也互斥
由概率加法公式：P(B) = P(BA₁) + P(BA₂) + … + P(BAₙ)
由乘法公式：P(BAi) = P(Ai) · P(B|Ai)
代入得：P(B) = Σ P(Ai) · P(B|Ai)   ✓
